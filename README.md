# Denning Chat

[Denning Chat](https://52.40.151.221/chat) is an open source code chat application with wide range of communication features available (such as one-to-one messaging, group chat messaging, file transfers, notifications, Facebook signup and audio/video calls).

We are please to present you with an out of the box chat application. You can customize this application depending of your needs. [QuickBlox](https://quickblox.com) is used for the backend.
